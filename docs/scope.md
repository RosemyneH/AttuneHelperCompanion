# Scope

Attune Helper Companion is a native C companion app for Synastria players.
It manages addons and tracks AttuneHelper daily snapshot history from local SavedVariables files.
The app does not download client files, manage user accounts, or store user files remotely.
Local WTF and AddOns backups are stored under the selected Synastria folder in AttuneHelperBackup.
Addon distribution starts with static manifests and ZIP packages.
